# StudyBuddy AI — Landing Page

A modern, fully responsive landing page built for **StudyBuddy AI**, a fictional AI-powered study companion concept. This project was built as Task 1 of my Full Stack Web Development internship, with a focus on writing clean semantic HTML and modern CSS from scratch — no frameworks, no JavaScript.

## Overview

StudyBuddy AI helps students turn their notes into instant quizzes and flashcards. This landing page introduces the product with a clear hero section, a feature breakdown, and a simple, distraction-free layout designed to feel modern and trustworthy.

## Features

- **Sticky navigation bar** with a CSS-only dropdown menu under "Services"
- **CSS-only hamburger menu** for mobile — no JavaScript required, built using the checkbox hack
- **Gradient hero section** with a highlighted heading and a call-to-action button
- **Feature cards** with icons, hover animations, and a responsive auto-adjusting grid
- **Fully responsive design** across desktop, tablet, and mobile screen sizes
- Smooth hover transitions and shadow effects throughout for a polished, modern feel

## Built With

- **HTML5** — semantic structure
- **CSS3** — Flexbox, Grid, media queries, pseudo-selectors
- **Font Awesome** — icons
- **Google Fonts** — Montserrat & Space Grotesk

## What I Learned

- Structuring semantic, accessible HTML
- Building a CSS-only dropdown and hover-based navigation
- Debugging real-world CSS issues (unclosed braces, selector typos, hover timing gaps)
- Implementing responsive design using media queries and CSS Grid's `auto-fit`/`minmax`
- Creating a mobile navigation menu without relying on JavaScript
